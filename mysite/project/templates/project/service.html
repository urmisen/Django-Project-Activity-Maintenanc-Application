SERVICES

Activity Maintenance Application is a web based application developed on python platform. This is for everyday use for student and teacher of a university. This app serves the used based on their role. There are mainly 3 different roles categorized as Admin, Student and Teacher. Based on the particular role’s operations/functions are divided uniquely.


MODULES :

1 : Admin
2 : Teacher
2 : Student

ADMIN :

In this Activity Maintenance Application, administrator can add or register all students and teachers in the University. Admin is responsible to maintain the database en which teacher/student information is present.


TEACHER :

• Teachers also need to get registered first. After getting the registration he/she can login into their accounts.

• Teachers can handle their profile and the personal data.

• Teachers can add Class Test marks according to course number and course name.

• They can edit the uploaded Class Test Data.

• Teachers can also delete data if needed.


STUDENT :

• Students need to get registered first. After getting the registration he/she can login into their accounts.

• Students can handle their profile .

• Students can view their Class Test marks according to course number and course name.
